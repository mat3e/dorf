Interface IDorfFieldDefinition<T>

Basic definition type for a field from Domain Object.

Every custom field should be built starting from the extension of this interface. It contains parameters to be defined for a field.

example

 //
 export interface IStarRatingDefinition<T> extends IDorfFieldDefinition<T> {
   max: number;
 }
